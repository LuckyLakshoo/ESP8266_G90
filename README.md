## About The Project

I have several devices I want to turn off using my SmartHome setup (Homebridge). As these devices are not smart, I came up with the idea to build some device that I can trigger with homebridge to press a button.

<p align="right">(<a href="#top">back to top</a>)</p>



### Built With

* [ESP8266 WIFI kit](https://www.espressif.com/en/products/socs/esp32)
* [G90 Servo](http://www.ee.ic.ac.uk/pcheung/teaching/DE1_EE/stores/sg90_datasheet.pdf)
* Battery, still resarching which battery can be used..
#* [Battery](https://vuejs.org/)
